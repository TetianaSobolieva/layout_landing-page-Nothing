Landing Page Nothing
A responsive landing page for Bose — built with pure HTML and SCSS, bundled with Parcel.

🔗 Demo
[DEMO LINK](https://tetianaSobolieva.github.io/layout_landing-page-Nothing/)

🛠 Tech Stack
HTML5 — semantic markup
CSS3 / SCSS — styling with variables, nesting and partials
Parcel — module bundler and dev server

🚀 Getting Started
Prerequisites
Node.js v14+
npm
Installation

# Navigate to the project folder
cd landing-page

# Install dependencies
npm install
Running locally
npm start
The app will open at http://localhost:1234

Build for production
npm run build
